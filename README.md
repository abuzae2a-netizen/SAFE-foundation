# SAFE Foundation Website

Complete, responsive website for SAFE Foundation and SAFE Council.

## Features

✅ Hero section with Call-to-Action buttons  
✅ 5 Mission cards  
✅ 6 Projects grid  
✅ SAFE Council overview  
✅ Donation paywall system  
✅ 3 Interactive forms  
✅ Mobile responsive design  
✅ Smooth animations  

## Live Site

**GitHub Pages:** https://abuzae2a-netizen.github.io/SAFE-foundation/

## Setup

1. Clone the repository
2. Open `index.html` in a browser
3. Or deploy to web hosting

## Files

- `index.html` - Main HTML structure
- `css/styles.css` - All styling
- `js/script.js` - JavaScript functionality

## Customization

To customize:

1. Edit colors in `css/styles.css` (CSS variables)
2. Edit content in `index.html`
3. Modify JavaScript functions in `js/script.js`

## Deployment

### Netlify
1. Connect GitHub repo
2. Deploy automatically
3. Add custom domain

### GitHub Pages
1. Settings → Pages
2. Deploy from main branch
3. Add custom domain
